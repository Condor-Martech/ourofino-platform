# Projeto Ouro Fino: Plataforma Digital Modular

## Visão Geral

Este repositório contém a documentação completa e o código-fonte para a nova plataforma digital da Ouro Fino. O objetivo é criar uma solução modular, escalável e orientada a dados, integrando institucional, e-commerce, vendas de ingressos (Estância) e backoffice operacional.

## Stack Tecnológica

### Frontend
- **Framework**: Astro
- **Estilização**: TailwindCSS
- **Animações**: Framer Motion
- **Linguagem**: TypeScript

### Backend & Dados
- **CMS**: Payload CMS
- **Banco de Dados**: PostgreSQL
- **Cache**: Redis
- **Infraestrutura**: Node.js, S3-compatible storage
- **Monitoramento**: Sentry

### Analytics
- Google Tag Manager (GTM)
- PostHog

## Objetivos de Negócio

1.  **Módulo Institucional**: Fortalecer a presença digital da marca e otimizar SEO.
2.  **Módulo Ecommerce**: Venda direta de produtos com experiência de usuário superior.
3.  **Módulo Estância**: Gestão eficiente de ingressos, reservas e controle de acesso via QR Code.
4.  **Backoffice**: Centralização operacional e eficiência administrativa.
